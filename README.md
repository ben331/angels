Angels
Este proyecto tiene como objetivo modelar una legion de arcangeles
para llevar un registro de ellos a modo de base de datos, y poder
aceder facilmente a la información de los arcangeles registrados.

El programa tiene 7 Funciones.

1.Add an archangel.
2.Remove an archangel.
3.Show and Count angels entered.
4.Search archangel by his name.
5.Search archangel by his power.
6.Show celebrations in the month.
7.Show all celebrations
8.Exit"

1.Add archangel. digitas los datos de un arcangel a guardar.
ADVERTENCIA:
	*No puedes guardar más de arcangeles al mismo tiempo.
	*El nombre del ángel debe terminar en "el".
	*El nombre del ángel no puede estar repetido.
	*El poder del ángel no puede estar repetido.

2.Remove an archangel.
Ingresa el nombre de un arcangel para eliminarlo y tener más espacio.

3.Show and Count angels entered.
Observa cuántos y cuales ángeles has ingresado.

4.Search archangel by his name.
Digita el nombre de un angel ingresado para ver su información.

5.Search archangel by his power.
Digita el nombre de un angel ingresado para ver su información.

6.Show celebrations in the month.
Selecciona un més del año para ver las celebraciones de ese més.

7.Show all celebrations
Observa las celebraciones de todos los ángeles ingresados.
